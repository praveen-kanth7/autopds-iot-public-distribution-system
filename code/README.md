AutoPDS: An IoT-Enabled Automated Public Distribution System

> Automating transparent, accurate, and efficient ration distribution through Embedded Systems and IoT.

📖 Overview

AutoPDS is an embedded IoT-based smart ration dispensing system designed to modernize the Public Distribution System (PDS). The system automates beneficiary authentication, accurate weighing, ration dispensing, and inventory monitoring to improve transparency, efficiency, and reduce manual errors.

✨ Features

- RFID-based beneficiary authentication
- Automatic ration dispensing
- Load cell-based weight measurement
- Servo-controlled dispensing mechanism
- LCD-based user interface
- Real-time inventory monitoring
- Cloud connectivity for data management
- Transparent digital transaction records

🛠️ Hardware Components

- ESP32
- RC522 RFID Reader
- HX711 Load Cell Amplifier
- Load Cell
- Servo Motor
- LCD Display
- Power Supply

💻 Software Used

- Arduino IDE
- Embedded C/C++
- Git & GitHub
